## Installation

### Prerequisites

- Python >= 3.10
- [pipx](https://pipx.pypa.io/stable/installation)
- [FiraCode Nerd Font](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/FiraCode.zip)

```sh
pipx install setch-ext
```

## Usage

```sh
setch
```
